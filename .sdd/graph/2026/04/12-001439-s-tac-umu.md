Implemented pre-flight validator per plan d-tac-hjp. (1) Prompt templates: 7 files in sdd/preflight_templates/ — 6 check-type templates (closing_action, closing_decision, decision_refs, action_closes_signals, signal_capture, supersedes) plus contracts.tmpl partial included via template directive. Embedded via go:embed with text/template. (2) Pre-flight library in sdd/preflight.go: PreflightRunner interface, CheckType enum with String(), SelectCheckType, AssembleContext, FormatEntryForPrompt, RenderPrompt (parses all templates together for partial resolution), ParseResult, RunPreflight orchestrator. No os/exec imports, all dependencies injected. Deviation: no PreflightFailedError type — design dialogue concluded the library returns PreflightResult with Pass/Gaps and errors only for infra failures; CLI handles fail results directly, typed error added no value since CLI is the sole consumer. (3) CLI integration: claudeRunner, --skip-preflight flag (preflight:skipped annotation), --preflight-model flag (default haiku), wired between ValidateEntry and file write. (4) Hard failure on all errors including infra: 120s timeout, claude not found or timeout rejects entry creation, user must --skip-preflight to bypass. Deviation from plan item 4 which specified graceful degradation with preflight:error annotation — user feedback during implementation: silent annotation is not obvious degradation, fail hard and let the user decide. Timeout changed from plan's 30s default to 120s based on real-world testing showing plan-closing prompts need more time. (5) Unit tests: TestSelectCheckType (12 cases), TestCheckTypeString, TestFormatEntryForPrompt (3), TestAssembleContext (7), TestRenderPrompt (all types + invalid + conditional), TestParseResult (11), TestRunPreflight (4 scenarios). All deterministic with mock runner. Deviation: no TestPreflightFailedError since the type was not implemented per item 2 deviation. (6) Eval tests behind //go:build eval: 6 golden cases — missing plan items, full coverage different wording, smuggled decision, valid signal, real graph history a-tac-tsd scope-out, contract violation. All 6 pass.
