Three reference fields with distinct semantics: refs (context/builds on, no status effect), supersedes (replaces — target no longer active/open, any type can supersede same type), closes (resolves/fulfills — target no longer active/open, decisions close signals, actions close decisions and signals). Open signal = not superseded, not closed. Active decision = not superseded, not closed. Note: s-cpt-jb3 should be closed by this decision once the closes field is implemented.
