Reframed signal_capture.tmpl type-correctness check (commit e80cc3e). Previous wording rejected signals containing solution-space thinking as 'smuggled decisions.' New wording uses the dialogued framing: signals expand dialogue by bringing observations and potential directions as material for discussion; decisions resolve dialogue into commitments. The test is now whether the entry prescribes a commitment or explores material for dialogue. No regression test: this is a prompt template wording change, not a code bug — d-prc-31v applies to code fixes, not prompt refinements.
