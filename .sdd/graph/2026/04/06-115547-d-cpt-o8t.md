The framework uses an immutable decision graph implemented on Git. Three types (signal, decision, action), five layers (strategic, conceptual, tactical, operational, process). Documents are never modified — superseded by new decisions. Current state is always reconstructed by traversing the graph, never maintained separately. Each fact lives in exactly one place.
