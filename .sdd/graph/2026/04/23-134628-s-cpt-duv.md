Multilingual SDD infrastructure landed per the plan (d-cpt-x38): per-graph `language` in `.sdd/config.yaml` governs entry authoring, the `/sdd` skill renders translated vocabulary on demand, pre-flight enforces language drift, and German is the first bundled locale. Implementation in commit 7477d65; end-to-end smoke verified with `language: de` — a German signal passes pre-flight, an English signal triggers `[high] language-drift` and blocks.

AC coverage:

- **AC1 — config field**: `model.Config.Language` added as top-level `language` YAML key; `MergeConfig` merges it; `FormatConfig` includes a commented example. Empty means English default.
- **AC2 — status header**: `query.StatusResult.Language` populated by `finders.Finder.language()`; `presenters.RenderStatus` emits `Language: <locale>` between `Local participant:` and `Graph:` when non-empty (suppressed for default).
- **AC3 — German vocabulary**: `internal/bundledskills/claude/sdd/references/vocabulary-de.md` created covering 2 types, 10 kinds, 5 layers, status notation, ref-field semantics, confidence levels, and common rendering terms, with explicit "never translated" boundaries for YAML frontmatter, CLI tokens, and entry IDs.
- **AC4 — on-demand vocabulary**: New `### Language — render translated vocabulary, author entries in the configured language` section in SKILL.md directs the agent to read `references/vocabulary-<locale>.md` on demand when `Language:` names a non-English locale and use translated terms in user-facing rendering (catch-up, playback, status narration, grooming tables).
- **AC5 — capture-time canonicalization**: Same SKILL.md section specifies that captured descriptions are authored in the configured language, while dialogue flows freely in any language; the skill translates dialogue content at `sdd new` time.
- **AC6 — pre-flight drift check**: `internal/llm/preflight_templates/language.tmpl` added as a shared partial with binary severity (pass or `high`) and calibration allowing mixed prose with English technical terms. Wired into all eight capture templates alongside `participants`. `ConfiguredLanguage` threaded through `preflightContext`, `Preflight()`, `assembleContext()`, and `finders.Preflight()` via `finder.language()`; empty disables the check.
- **AC7 — end-to-end smoke**: verified with `language: de` temporarily in `.sdd/config.yaml`. German signal `--dry-run`: exit 0, no findings. English signal `--dry-run`: exit 1, one `[high] language-drift` finding naming the primary language and configured language. Config reverted to English default after verification.
