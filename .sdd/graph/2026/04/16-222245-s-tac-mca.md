Implements short-form entry ID acceptance across the sdd CLI, closing plan d-tac-g8n and signal s-prc-b2v.

Change surface (commits 57d636a, ed0f267):
- `model.Graph.ResolveID` / `ResolveIDs`: maps `{type}-{layer}-{suffix}` to the full ID on unique match; errors listing all candidates on ambiguity; passes input through unchanged when the shape is unrecognized, type/layer abbrev is unknown, or zero matches exist — so "entry not found" stays the caller's existing surface.
- Wired into `finders/show.go`, `handlers/handler_summarize.go`, and `handlers/handler_new_entry.go` (refs/closes/supersedes resolved before validation, so the handler operates on full IDs throughout).
- `/sdd` skill text + `references/cli-reference.md`: agents keep invoking the CLI with full IDs, stated once as a single clear rule covering all ID arguments; short IDs remain fine in user-facing narrative.
