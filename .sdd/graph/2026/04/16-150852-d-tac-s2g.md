Add sdd init command and consolidate SDD metadata under .sdd/ at the repository root. sdd init interactively prompts for graph dir (default .sdd/graph) using charmbracelet/bubbles textinput, creates the directory structure, writes config.yaml from a commented template, adds .sdd/tmp to .gitignore, and creates the graph dir. Bubbles is also the TUI library for future interactive prompts in sdd install (agent target selection, install scope per d-cpt-uu1). Without init, CLI uses convention-based defaults. The CLI auto-discovers .sdd/ by walking up from cwd. .sdd/tmp/ is auto-created on first use. Migrate existing .sdd-tmp/ from graph dir.
