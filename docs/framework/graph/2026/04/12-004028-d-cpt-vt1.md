Agent reliability is achieved through structural separation — tool-layer validation by independent agents — not through instructional improvement. The agent that acts cannot reliably certify its own compliance. All reliability-critical workflows must include a validation step performed by a separate agent or tool invocation. Self-monitoring is advisory input, never the sole gate.
