Agent reliability for critical process steps is achieved through structural separation — tool-layer validation by independent agents. The agent that acts cannot reliably certify its own compliance. All reliability-critical workflows must include a validation step performed by a separate agent or tool invocation. Self-monitoring is advisory input, never the sole gate. Instructional guidance (skills, prompts) remains valuable for judgment and heuristics, but is not sufficient as the sole enforcement mechanism for compliance.
