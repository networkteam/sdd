Plan for --attach filename mapping. (1) Change --attach from StringFlag to repeatable flag (urfave/cli v3 StringSliceFlag). Remove comma-splitting. (2) Parse source:target syntax — split on last colon. If no colon, target = filepath.Base(source). (3) Implement - stdin alias — read stdin, write to target name. - requires :target, at most one - per invocation. (4) Update file copy logic — use target name for destination and entry.Attachments list. (5) Tests: mapping, basename fallback, stdin, bare - error, duplicate - error. (6) Update /sdd skill — add --attach syntax to SKILL.md and framework-concepts.md with examples. (7) Rebuild binary (not committed).
